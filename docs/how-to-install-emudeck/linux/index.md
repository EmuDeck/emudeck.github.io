## Prerequisites 

### Ubuntu/Debian

sudo apt-get install steam jq zenity flatpak unzip bash

### Arch

sudo pacman -Sy packagekit-qt5 flatpak rsync unzip jq bash curl zenity

### Fedora

dnf install flatpak unzip jq bash steam
