## Prerequisites 

### Arch

`sudo pacman -Sy bash curl flatpak jq packagekit-qt5 rsync steam unzip zenity`

### Debian/Ubuntu

`sudo apt-get install bash curl flatpak jq rsync steam unzip zenity`

### Fedora

`dnf install bash curl flatpak jq rsync steam unzip zenity`
