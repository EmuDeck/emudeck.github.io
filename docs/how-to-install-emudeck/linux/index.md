## Prerequisites 

### Arch

`sudo pacman -Sy steam packagekit-qt5 flatpak rsync unzip jq bash curl zenity`

### Fedora

`dnf install flatpak unzip jq bash steam zenity rsync curl`

### Debian/Ubuntu

`sudo apt-get install steam jq zenity flatpak unzip bash rysnc curl`
